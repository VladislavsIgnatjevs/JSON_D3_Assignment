Preparing stuff

download this zip archive:
ne_10m_admin_0_map_subunits.zip

First need to install gdal and topojson:
Easiest way is to install using Homebrew
'brew install gdal'
'brew install npm'
and then
'npm install -g topojson'

verify:
'which ogr2ogr'
'which topojson'

..should return /usr/local/bin/ogr2ogr and /usr/local/bin/topojson

Now, for simplicty, lets install and run http server:
'npm install http-server -g'

then run 'http-server -p 8008 &'

unzip archives into your project dir.

now 'cd ne_10m_admin_0_map_subunits' and run the following:

ogr2ogr /
-f GeoJSON /
-where "ADM0_A3 IN ('GBR', 'IRL', 'IMN')" /
countries_regions.json /
ne_10m_admin_0_map_subunits.shp

this will create geojson format json file (countries_regions.json)

Options like 'IMN', 'GBR' can be added or removed according to ISO 3166-1 alpha 3

then 'mv countries_regions.json ..'
'cd ..' will get you back to your project dir.
run the following here:

topojson /
-o ukCountries.json /
--id-property SU_A3 /
--properties name=NAME /
--countries_regions.json

this will create topojson format json file (countries_regions.json) that is used in a project
